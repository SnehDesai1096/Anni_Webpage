# Anni_Webpage

## Description
Anni_Webpage is a heartfelt and interactive webpage created to celebrate a first wedding anniversary. The webpage is designed to convey love and emotions through animations, music, and a countdown timer that tracks the time spent together.

## Features
- **Responsive Design**: Ensures the webpage looks great on all devices.
- **Interactive Music Player**: Background music that can be toggled on or off.
- **Countdown Timer**: Displays the time spent together in days, hours, minutes, and seconds.
- **Animations**: Smooth fade-in effects and floating clouds to enhance the visual appeal.
- **Custom Messages**: Personalized quotes and messages to express love.

## Technologies Used
- **HTML5**: For the structure of the webpage.
- **CSS3**: For styling, animations, and responsive design.
- **JavaScript**: For interactivity, including the countdown timer and music player.

## How to Use
1. Open the `index.html` file in any modern web browser.
2. Enjoy the background music and animations.
3. Toggle the music on or off using the button at the bottom right corner.

## Files
- `index.html`: The main HTML file for the webpage.
- `styles.css`: Contains all the styles and animations.
- `script.js`: Handles the interactivity, including the countdown timer and music player.
- `music.mp3`: Background music file (ensure it is in the same directory).
